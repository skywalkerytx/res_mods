Поздравляю, Дружище!

Вы используете моды, которые входят в состав Мультипака от PROТанки: http://www.youtube.com/user/PROTankiWoT
Команда Мультипака поддерживает всех авторов модов и сборок модов (модпаков)!
Вы можете использовать любые моды из сборки для создания собственных модов на их основе. Если Вам необходима помощь, то Вы можете связаться со мной: protanki@bk.ru
Вы можете включать моды из Мультипака в собственные сборки модов, но не забывайте указывать источник и авторов: ProTanki, P0LIR0ID.

САМОЕ ГЛАВНОЕ: Для использования модов НЕЛЬЗЯ удалять данный файл README.txt и менять его местоположение!

КАК РЕДАКТИРОВАТЬ КОНФИГУРАЦИИ МОДОВ?

Вы уже заметили, что конфигурации для модов имеют незнакомое Вам расширение: *.ptc
Это специальное расширение для модов от PROТанки - Pro Tanki Configuration
Для редактирования используйте только Notepad++ или ему подобные редакторы.
Скачать можно тут: http://notepad-plus-plus.org/download 
Не используйте стандартный блокнот Windows. После правок блокнотом, мод перестанет работать.

КАКИЕ МОДЫ РАСПРОСТРАНЯЮТСЯ НА ДАННУЮ ЛИЦЕНЗИЮ? ДЛЯ ЧЕГО ОНИ НУЖНЫ?

ProSights.pyc - это универсальный скрипт для прицелов.
Данный скрипт необходим для прицелов, которые показывают информацию о противнике, бронепробитие и толщину брони, перезарядку в чате и многое другое.
Скрипт был успешно проверен на самых популярных прицелах, подтвердил свою работоспособность и скорость работы.
Вы можете использовать данный скрипт при создании собственных прицелов, рассчитывать на нашу поддержку.
Данный скрипт может исправить под патч 0.9.2 любой старый прицел, для этого необходимо отредактировать файлы в папке: /ProSights
arcade.ptc - аркадный прицел;
sniper.ptc - снайперский прицел;
postmortem.ptc - прицел после уничтожения танка;
strategic.ptc - артиллерийский прицел.
reducedArmor.ptc - включает и выключает улучшенный маркер бронепробития. Значения: off - выключено, cut -  только цвета индикатора, full - полная версия.

ProDamageAnoncer.pyc - оптимизированный мод, который выводит сообщения в боевой чат о стрелявшем по Вам противнике и времени его перезарядки.
ProDamageAnoncer.ptc - файл с конфигурацией данного мода.

ProSound.pyc - мод для замены оригинальных звуковых событий в игре.
CriticalSound.ptc - файл с конфигурацией для настройки путей к альтернативным звукам повреждений танка - "Звоночек крита"
EnemySigned.ptc - файл с конфигурацией для активации голосового уведомления о засветке противника! "Противник обнаружен"
FireFirst.ptc - файл с конфигурацией для изменения звука и приоритет сообщения о возгорании танка. "Танк горит" - имеет самый высокий приоритет оповещения.

DamageIndicator.pyc - мод увеличивает время отображения маркера направления атаки врага в 4 раза.

ProVoiceController.pyc - мод для голосовой активации микрофона в игре
ProVoiceController.ptc - файл с конфигурацией данного мода.

ProSpotted.pyc - мод для оповещения союзников о засвете.
ProSpotted.ptc - файл с конфигурацией данного мода.

PROHangarChoicer - мод для смены ангаров при активации ивентов ангаров от WG.

ProStatControl.pyc - мод для смены отображаемой статистики XVM прямо в ангаре.
